# アクセシビリティ

Trial 003ではPlaywright E2Eのaxe検査で `color-contrast` を有効化しています。ホーム、検索、動画詳細、状態表示の基本ページは、色コントラストを含めてaxe違反ゼロを目標にします。

## 対応

- 暗色のプレイヤー操作領域では、補助テキストの色を明るくしてコントラストを確保します。
- ホームの通知、カテゴリ、サイドナビ、ダッシュボードは既存トークンを使い、文字色と背景色の組み合わせを限定します。
- アイコンのみのボタンには `aria-label` を付けます。
- 動画プレイヤーのエラー領域は `role="alert"` で通知します。

## 除外

Trial 003時点でaxeの `color-contrast` ruleは除外していません。将来、動画サムネイル上の一時的なオーバーレイなどで合理的な除外が必要になった場合は、対象要素と理由をこの文書に追記します。
