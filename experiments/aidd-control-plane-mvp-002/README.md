# AIDD Control Plane MVP 002

## 目的

MVP 001で作った「Product Brief -> AI Task Packet -> Agent Runbook -> Review -> Learning Log」の画面に、AIDD-Spec v0.1準拠のJSON Schema + Contract Checkerを追加する。

## 今回の範囲

- AI Task Packet JSONの入力/編集
- Verification Evidence JSONの入力/編集
- Review Record JSONの入力/編集
- Learning Log JSONの入力/編集
- JSON Schema風の必須項目検査
- 合否サマリー
- 不足項目リスト
- 改善提案
- サンプルJSON挿入
- 画像つき記事化用キャプチャ

## 非ゴール

- 外部API送信
- DB保存
- ログイン
- 課金
- 本番SaaS化
- 完全なJSON Schema Draft実装

## 準拠目標

AIDD-Spec v0.1 L3。
